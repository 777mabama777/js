Teoria JavaScript

1. Cztery sposoby wyprowadzania danych w JS:

1) document.write() - wyświetlanie bezpośrednio na stronie

2) alert() - wyświetlanie w okienku alert

3) Wyprowadzanie do formularzy - przez pola input

4) Wyprowadzanie do tabeli - przez elementy table

2. Funkcja w JavaScript to:

  • Zbiór zgrupowanych instrukcji uruchamianych przez nazwę funkcji
  • Wykonuje wewnętrzny kod po wywołaniu
  • Może przyjmować argumenty do przetwarzania (np. suma(a,b))
  • Może zwracać wartość przez słowo kluczowe return

3. Metody pobierania elementów:

Metoda Opis
getElementById("id") Dostęp do elementu o określonym ID
getElementsByName("nazwa") Dostęp do elementów o określonej nazwie
getElementsByTagName("znacznik") Dostęp do wszystkich elementów danego typu (np. p, div)

4. Właściwości elementów:

innerHTML: Umożliwia odczyt i modyfikację zawartości HTML elementu

innerText: Pozwala na ustawienie lub odczyt tekstu w elemencie bez formatowania HTML

className: Umożliwia zmianę lub odczyt klasy CSS elementu
